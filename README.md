### Installation

Install with npm

To get specific version

```bash
  npm install @osnmarket/prunella@<version>
```

To get lastest version

```bash
  npm install @osnmarket/prunella@lastest
```

### or

Install with yarn

To get specific version

```bash
  yarn add @osnmarket/prunella@<version>
```

To get lastest version

```bash
  yarn add @osnmarket/prunella@lastest
```
